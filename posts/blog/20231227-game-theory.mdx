## 게임 이론

두 플레이어가 순서와 상관 없이 움직일 수 있는 방법이 동일하게 정해져 있고, 무작위적인 요소가 없는 게임을 번갈아 가며 진행합니다.

각 플레이어는 항상 최적의 방법으로 게임을 진행하여 최대한 좋은 결과를 얻기 위해 노력합니다.

`게임 이론`은 이러한 게임에서 상대방이 어떤 움직임을 선택하더라도 항상 이길 수 있는 전략이 있는 경우가 있는지 살펴보는 방법입니다.

## 게임 상태

`승리 상태`는 최적의 방법으로 게임을 진행할 때 승리할 수 있는 상태를 의미하며, `패배 상태`는 상대방이 최적의 방법으로 진행한다면 패배하는 상태를 말합니다.

현재 상태에서 패배 상태로 이동할 수 있다면 승리 상태가 되며, 그렇지 않다면 패배 상태가 됩니다.

### 돌 게임

https://www.acmicpc.net/problem/9655

각 플레이어가 돌을 1개 또는 3개를 가져갈 때, 마지막으로 돌을 가져가는 플레이어가 승리합니다.

남아있는 돌의 개수는 게임의 상태에 대응됩니다.

처음 돌의 개수가 0개 있다고 하면, 선공 플레이어는 항상 패배하므로 n = 0은 패배 상태가 됩니다.

n = 1일 때, 선공 플레이어는 돌을 전부 가져가서 패배 상태(n = 0)로 이동시킬 수 있으므로 승리 상태가 됩니다.

n = 2일 때, 선공 플레이어가 돌을 1개 가져가면 돌이 하나 남게되고 게임의 상태는 n = 1로 이동하게 됩니다. 즉, 패배 상태로 이동하지 않으므로 패배 상태가 됩니다.

마찬가지로 n = 3, 4, .. 일 때 게임의 상태가 패배 상태로 이동할 수 있는지 확인해보면 처음 돌의 개수만 봐도 승리할 수 있는지 알 수 있습니다.

```cpp
if (n % 2 == 0) cout << "CY";
else cout << "SK";
```
