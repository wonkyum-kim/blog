## mask-image

`clip-path`가 엘리먼트를 잘라내는 것이라면, `mask-iamge`는 엘리먼트를 이미지나 그라데이션과 겹치는 역할을 한다.

## 이미지로 마스킹

두 이미지를 겹치는 방법

그냥 겹쳐진 하나의 이미지를 만들어 오는 것이 더 나을듯

## SVG로 마스킹

이미지를 겹치는 것보다야 낫겠지만, 그냥 이미지를 만들어 오는 것이 더 나을 것 같음

## 그라데이션 마스킹

가장 유용하게 쓸 수 있을 듯

.container img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  -webkit-mask-image: linear-gradient(to top, transparent 5%, black 100%);
  mask-image: linear-gradient(to top, transparent 5%, black 100%);*/
}
