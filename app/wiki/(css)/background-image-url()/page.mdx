## background-image: url()

그냥 사용하면 LCP에 영향을 미치기 때문에 조심한다.

아래와 같은 최적화 기법과 함께 사용한다.

### 안에 보이지 않는 img 넣기

<CodeSelector names={['html']} desc='img' />

```html
<div style="background-image: url(lcp.jpg)">
  <img src="lcp.jpg" alt="" width="0" height="0" style="display: none !important;" />
</div>
```

### rel=preload

`link` 엘리먼트의 `rel` 어트리뷰트는 HTML `head`에서 브라우저가 렌더링을 시작하기 전에 페이지에서 곧 필요한 리소스에 대한 fetch 요청을 할 수 있게 해준다.

이렇게 하면 더 일찍 리소스를 사용할 수 있고 페이지 렌더링을 차단할 가능성이 낮아져 성능이 향상된다.

<CodeSelector names={['html']} desc='preload' />

```html
<link rel="preload" as="image" href="lcp-image.jpg" />
```
