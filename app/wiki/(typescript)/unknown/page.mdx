## unknown

`unknown`은 `any`와 비슷하지만 좀 더 안전하다.

any는 에러를 발생 시키지 않지만 unknown은 에러를 발생 시킨다.

<CodeSelector names={['ts']} desc='anyunknown' />

```ts
function foo1(callback: any) {
  callback()
}

function foo2(callback: unknown) {
  callback()
}

foo1('hi') // 타입스크립트에서 에러가 발생하지 않는다.
foo2('hi') // 타입스크립트에서 에러가 발생
```

unknown을 사용하였다면 타입 좁히기를 사용하여 의도에 맞게 사용하면 된다.

<CodeSelector names={['ts']} desc='as' />

```ts
function foo(callback: unknown) {
  if (typeof callback === 'function') {
    callback()
  }
}
```
