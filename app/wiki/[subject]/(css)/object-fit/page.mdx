## object-fit

이미지와 컨테이너의 비율이 다를 때 이미지의 크기를 조절하는 프로퍼티.

- cover: 이미지 비율을 유지한 채로 컨테이너에 넣는다. 컨테이너와 이미지의 비율이 일치하지 않으면, 이미지가 잘려나간다.
- contain: 이미지 비율을 유지한 채로 컨테이너에 넣는다. 레터박스가 생길 수 있다.
- fill: 이미지가 컨테이너의 크기에 맞게 조절된다. 컨테이너와 이미지의 비율이 일치하지 않으면, 이미지의 비율이 변경된다.
- none: 이미지의 크기를 조절하지 않는다. `object-position`과 같이 사용할 수 있다.
- scale-down: none과 contain 중, 이미지의 크기가 더 작아지는 것을 선택한다.

[object-fit 프로퍼티를 사용한 예시](https://developer.mozilla.org/ko/docs/Web/CSS/object-fit)

view transitions api를 사용할 때, view는 이미지이므로 비율이 달라진다면, object-fit을 사용할 수 있다.
