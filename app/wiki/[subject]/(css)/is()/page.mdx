## :is()

반복을 줄일 때 사용할 수 있다.

<CodeSelector names={['css']} desc='is' />

```css
/* before */
ul li,
ol li {
}

/* after */
:is(ul, ol) li {
}
```

마치 반복문을 사용하는 것과 같다.

<CodeSelector names={['css']} desc='is2' />

```css
:is(section, article, aside, nav) :is(h1, h2, h3, h4, h5, h6) {
  color: #bada55;
}

/* ... which would be the equivalent of: */
section h1,
section h2,
section h3,
section h4,
section h5,
section h6,
article h1,
article h2,
article h3,
article h4,
article h5,
article h6,
aside h1,
aside h2,
aside h3,
aside h4,
aside h5,
aside h6,
nav h1,
nav h2,
nav h3,
nav h4,
nav h5,
nav h6 {
  color: #bada55;
}
```
