## perspective

`perspective`란 멀고 가까운 거리에 대한 느낌을 의미함.

CSS에서는 z = 0인 평면과 사용자 간의 거리를 설정할 수 있다.

perspective를 작게 주면 커 보이고 크게 주면 작게 보인다.

## 첫 번째 방법

<CodeSelector names={['css']} desc='first' />

```css
.panel--red {
  transform: perspective(400px) rotateY(45deg);
}
```

`transform`의 perspective를 사용한다.

이 방법은 개별 요소마다 원근감을 주기 때문에 각각의 소실 점이 생겨서 여러 엘리먼트를 렌더링하는 경우에는 부적절하다.

## 두 번째 방법

<CodeSelector names={['css']} desc='second' />

```css
.scene--blue {
  perspective: 400px;
}

.panel--blue {
  transform: rotateY(45deg);
}
```

부모 요소에만 원근감을 부여해서 하나의 소실점을 가지도록 한다.
