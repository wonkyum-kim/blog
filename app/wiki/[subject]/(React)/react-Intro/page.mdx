## react 패키지

`react` 패키지는 React 컴포넌트를 정의하는데 필요한 기능만 포함되어 있다.

보통 웹을 위한 `react-dom` 또는 네이티브 환경을 위한 `react-native`와 같은 `React 렌더러`와 함께 사용된다.

일부 구현은 `react-reconciler`와 같은 다른 패키지에도 존재한다.

참고로 react 패키지뿐만 아니라 다른 모든 패키지에서 개발 버전에 대한 코드는 `__DEV__`로 관리되고 있기에, 생략하고 관찰하도록 한다.

`react` 패키지에서 export 하고 있는 모듈은 [`index.js`](https://github.com/facebook/react/blob/main/packages/react/index.js)에서 확인 가능하다.
