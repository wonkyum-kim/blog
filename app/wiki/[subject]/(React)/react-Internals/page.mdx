## ReactSharedInternals

`ReactSharedInternals`는 react 패키지 내부에서 사용하는 객체로 현재 코드에서는 비어있는 상태로 존재한다.

이 비어있는 객체는 `shared` 패키지의 ReactSharedInternals.js에서 import되고 다시 export되어 다른 모든 패키지에서 공유된다.

- ReactSharedInternals.H: dispatcher를 나타내며, Hook을 관리한다. `react-reconciler` 패키지에서 [주입](/wiki/React/react-reconciler-Hook)된다.
