## 오일러 경로

`오일러 경로`는 그래프의 각 간선을 정확히 한 번씩 지나는 경로다.

그러한 경로의 시작과 끝이 같은 노드인 경우를 `오일러 회로`라고 한다.

## 양방향 그래프

양방향 그래프에서 오일러 경로가 존재하는 경우는 모든 간선이 같은 연결 컴포넌트에 속하고 다음 두 조건 중 하나를 만족하면 된다.

1. 모든 노드의 차수가 짝수
2. 정확히 두 노드의 차수가 홀수이고, 다른 모든 노드의 차수가 짝수

첫 번째 경우, 오일러 경로는 오일러 회로가 된다.

두 번째 경우, 차수가 홀수인 두 노드가 오일러 경로의 양 끝 점이 되며, 오일러 경로는 오일러 회로가 아니다.

## 방향 그래프

방향 그래프에서 오일러 경로가 존재하는 경우는 모든 간선이 같은 강한 연결 컴포넌트에 속하고 다음 두 조건 중 하나를 만족하면 된다.

1. 모든 노드의 indegree와 outdegree가 같다.
2. 한 노드의 indegree가 outdegree보다 1보다 크고, 다른 한 노드의 outdegree가 indegree가 1보다 크며, 나머지 노드는 indegree와 outdegree가 같다.

첫 번째 경우, 오일러 경로는 오일러 회로가 된다.

두 번째 경우, outdegree가 큰 노드에서 시작해서 indegree가 큰 노드에서 끝나는 오일러 경로가 존재한다.

두 번째 조건을 만족하는 노드의 indegree가 1이고 outdegree가 0이거나, outdegree가 1이고 indegree가 0이면 강한 연결 요소에 속하지 않아도 오일러 경로가 존재할 수 있다.

## Hierholzer's algorithm

`Hierholzer's algorithm`은 그래프의 오일러 회로를 찾는 알고리즘이다.

시작 지점을 하나 정하여 dfs를 시작하고, 현재 정점과 다음 정점이 연결된 간선을 지우면서 dfs를 진행한다.

양방향 그래프의 경우 간선을 지우기 위해서 인접 리스트와 인접 행렬을 모두 사용한다.

```js showLineNumbers
// 양방향 그래프의 경우
function dfs(curr) {
  while (adj[curr].length) {
    const next = adj[curr][adj[curr].length - 1]
    adj[curr].pop()
    if (matrix[curr][next] && matrix[next][curr]) {
      matrix[curr][next]--
      matrix[next][curr]--
      dfs(next)
    }
  }
  ans.push(curr + 1)
}
```
