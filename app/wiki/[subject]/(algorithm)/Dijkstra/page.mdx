## 다익스트라 알고리즘

가중치가 음수가 아닌 경우 최단 경로를 구할 수 있는 알고리즘.

우선순위 큐를 사용하여 아직 처리하지 않은 정점 중에 거리가 가장 작은 정점을 찾고, 그 정점에서 시작하는 모든 간선을 살펴보며 거리를 줄일 수 있으면 줄인다.

이 알고리즘은 거리가 계속 증가한다고 가정하여 가장 최솟값을 선택하는 그리디 방법을 사용하고 있다.

따라서 이미 거리가 정해진 정점은 이미 최솟값으로 정해져 있는 상태이므로 다시 살펴볼 필요가 없다.

하지만 음수인 간선이 포함되는 경우 이 방법을 사용할 수 없다.

왜냐하면 지금은 거리가 최솟값이 아니지만 언젠가 음수인 간선을 만나서 최솟값을 선택하는 것보다 더 짧은 거리로 갈 수 있다고 한다면 그리디 방법을 쓸 수 없기 때문이다.

const dist = new Array(n).fill(Infinity)

function Dijkstra(start) {
  dist[start] = 0
  const pq = new PriorityQueue()
  pq.push([0, start])

  while (!pq.empty()) {
    const [a, curr] = pq.top()
    pq.pop()
    if (dist[curr] < -a) continue // 이미 처리된 정점
    for (const [next, w] of adj[curr]) {
      if (dist[curr] + w < dist[next]) {
        dist[next] = dist[curr] + w
        pq.push([-dist[next], next])
      }
    }
  }
}
